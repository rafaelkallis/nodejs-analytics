NodeJS Analytics

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. The Node.js package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Below you can find a network analysis of the Node.js github repository. The analysis is focused exclusively on github users which have interacted with the Node.js repository. Please note that data from not registered users is not included in the analysis.

Abstract overview over the collected data. In the core we have the contributors. Edges are directed and represent a "follows" relationship. In the outer ring, we have users which have a relationship to contributors, but didn't contribute to Node.js
